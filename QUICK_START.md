# ⚡ Quick Start Guide - K-Drive

## 🚀 Get Started in 30 Seconds

### 1. Your app is already running!
Open your browser and visit:
```
http://localhost:3000
```

### 2. Try These Features:

#### Upload a File
1. Click the **"Upload"** button (top right)
2. Drag & drop any file or click to browse
3. Click **"Upload"** to confirm
4. Your file appears in the grid!

#### Create a Folder
1. Click **"New Folder"** button
2. Type a name (e.g., "My Documents")
3. Click **"Create"**
4. Folder appears in the grid!

#### Navigate Around
- **Sidebar**: Click "My Drive", "Starred", "Recent", etc.
- **View Toggle**: Switch between grid 📊 and list 📋 views
- **Search**: Type in the search bar to find files

#### Try Actions
- **Star**: Click the ⭐ icon to favorite items
- **Menu**: Click ⋮ (three dots) for more options
- **Delete**: Send items to trash
- **Open Folder**: Click any folder to navigate inside
